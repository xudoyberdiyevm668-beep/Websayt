## 📖 Loyiha haqida

**BuyCars** — bu dunyo mashhur avtomobil brendlarini (BMW, Bugatti, Lamborghini, Ferrari) namoyish etuvchi zamonaviy veb-platforma. Bu mening **birinchi mustaqil web loyiham** bo'lib, HTML, CSS va JavaScript ko'nikmalarini amalda qo'llash maqsadida yaratilgan.

## ✨ Asosiy imkoniyatlar

- 🏎️ **Premium avtomobillar katalogi** — BMW, Bugatti, Lamborghini, Ferrari

## 🚀 Ishga tushirish

### 1. Reponi klonlash

```bash
git clone https://github.com/username/buycarsc.git
cd buycarsc
```

### 2. Brauzerda ochish

```bash
# index.html faylini to'g'ridan-to'g'ri oching
open index.html
```

> ⚡ Hech qanday server yoki paket kerak emas — sof statik sayt!
